EDAL

ncWMS and the Environmental Data Abstraction Layer libraries.

What is EDAL?

EDAL is the Environmental Data Abstraction Layer - a set of libraries which provide a data model for dealing with environmental datatypes, and numerous utility routines for reading, manipulating, and plotting this data, as well as serving it over a WMS.

Getting EDAL

The EDAL libraries are designed to be integrated using Maven, and can be downloaded or added as dependencies from the Maven Central repository. See the EDAL user guide for more information.

What is ncWMS?

ncWMS is a Web Map Service for displaying environmental data

Getting ncWMS

ncWMS is designed to run in a servlet container such as Tomcat or Glassfish. However, we do provide a standalone version which uses an embedded Winstone servlet container to allow users to quickly try it out.

Servlet container version

This is the standard version of ncWMS which most people will want to use it can be downloaded from here

Note that you will need to configure a security role within your servlet container to be able to use this.

Standalone version

The standalone version of ncWMS can be downloaded from here

Note that this version is designed to be run locally and has no security for any of the administrative functions. To run a publicly-available version of ncWMS, please use the servlet container version.

Using ncWMS

For information on how to use ncWMS, please consult the ncWMS User Guide
