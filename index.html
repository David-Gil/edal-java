What is ncWMS?

ncWMS is a Web Map Service for displaying environmental data

What is EDAL?

EDAL is the Environmental Data Abstraction Layer - a set of libraries which provide a data model for dealing with environmental datatypes, and numerous utility routines for reading, manipulating, and plotting this data, as well as serving it over a WMS.

Getting started with ncWMS

ncWMS is designed to run in a servlet container such as Tomcat or Glassfish. However, we do provide a standalone version which uses an embedded Winstone servlet container to allow users to quickly try it out.

Developing ncWMS

ncWMS can be extended in a number of ways for custom behaviour. These include:

  • Adding new colour palettes
  • Precisely defining new styles using custom SLDs
  • Adding readers for new data formats
  • Adding plugins to perform on-the-fly manipulation of existing data layers
